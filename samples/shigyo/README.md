# 士業サンプル（モバイルファースト）

## プレビュー方法

- `samples/shigyo/index.html` をブラウザで直接開いてください。
- 端末幅を 375px 付近にするとスマホ想定のレイアウトを確認できます。

## セクション構成

1. ヒーロー（事務所名 / タグライン / CTA）
2. 業務内容（サービスカード）
3. 選ばれる理由
4. 料金の目安
5. ご相談の流れ
6. 対応事例
7. よくある質問（アコーディオン）
8. 事務所概要
9. お問い合わせ（フォーム / 連絡先）
10. フッター

## 補足

- `shigyo.js` でモバイルメニュー、スムーススクロール、FAQの開閉、
  ページ上部へ戻るボタンの表示制御を行っています。
- 外部画像は使用していません。アイコンはインラインSVGです。
